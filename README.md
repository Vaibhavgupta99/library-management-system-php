# 📚 Library Management System

A complete web-based Library Management System built using PHP, MySQL, Bootstrap, and JavaScript.

## 🚀 Features

* Admin & User Login
* Book & Movie Management
* Membership Management
* Issue & Return System
* Fine Calculation (₹5/day)
* Reports (Active, Overdue, Master Lists)

## 🛠 Tech Stack

* PHP
* MySQL
* Bootstrap 5
* JavaScript

## ⚡ Run Locally

1. Install XAMPP
2. Move folder to `htdocs`
3. Start Apache & MySQL
4. Open `http://localhost/library`
